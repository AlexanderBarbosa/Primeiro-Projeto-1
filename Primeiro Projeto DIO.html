"Principais Tags HTML":

(Voltar)

Tags estruturais:

  • Tag < html>< /html>

    Esta Tag é o elemento principal que vai comportar toda a estrutura

  • Tag < head>< /head>

    Elemento que armazena metadados(dados sobre dados)

  • Tag < meta>

    São usadas para especificar um conjunto de caracteres, como descrição da página, palavra chave, autor de documento e configurações da janela de visualização

  • Tag < body>< /body>

    A tag body define o corpo do documento, o elemento body contém todo o conteúdo de um documento HTML

  • Tag < section>< /section>

    Tag define uma seção em um documento

  • Tag < article>< /article>

    Tag que define um conteúdo independente e completo de outros, artigos com conteúdos que fazem sentido por si só.

  • Tag < div>< /div>

    Tag define uma divisão ou seção em um documento HTML

Tags de cabeçalho:

    Os elementos HTML representam seis níveis de título de seção, sendo o < h1> é o nível mais alto e o < h6> é o mais baixo.

Tags de links:

    O que é um link em linguagem HTML

    Link, em inglês, significa ligação, elo ou conexão. sendo isso que os links fazem em HTML, eles conectam uma página da internet até outra, eles levam o usuário de um local a outro. Um link pode ser um uma página HTML, uma imagem, um e-mail, um endereço de FTP etc. Os links são, portanto, um endereço, uma referência de um 'local' para outro.

    As tags mais usadas são:

    < a> (Âncora): Essa tag cria links e permite vincular elementos dentro da mesma página ou a outros recursos externos. Alguns atributos comuns incluem: href: Define o URL do documento vinculado (pode ser um link para um site, um endereço de e-mail etc.).

    < Target: Identifica onde o link deve ser aberto (por exemplo, em uma nova janela ou na mesma janela). rel e rev: Definem os tipos de link que avançam ou revertem a ação1.

    < link>: Embora não seja uma tag de link direto, a tag < link> é usada para vincular recursos externos, como folhas de estilo CSS, ícones e fontes. Ela é frequentemente encontrada no cabeçalho da página.

    < nav> (Navegação): Essa tag é usada para agrupar elementos de navegação, como menus e links de página. É uma boa prática usar essa tag para melhorar a acessibilidade e a semântica do seu site.

    < button> (Botão): Embora não seja exclusivamente para links, a tag < button> é frequentemente usada para criar botões de ação, como “Enviar” ou “Saiba mais”. Você pode adicionar eventos de clique a esses botões para executar ações específicas.

    < ul> (Lista não ordenada) e < ol> (Lista ordenada): Essas tags são frequentemente usadas para criar listas de links. A < ul> cria uma lista não ordenada com marcadores, enquanto a < ol> cria uma lista ordenada com números ou letras.

Tags de listas:

    As tags HTML são fundamentais para estruturar e formatar páginas da web. Elas demarcam elementos e fornecem informações ao navegador sobre a estrutura do site. Vamos explorar algumas das principais tags de listas que são frequentemente utilizadas: < ul> (Lista não ordenada) e < ol> (Lista ordenada):

    A tag < ul> é usada para criar listas não ordenadas (com marcadores), enquanto a < ol> é usada para criar listas ordenadas (numeradas). Ambas são usadas com a tag < li> para definir cada item da lista.

    < dl> (Lista de definições):

    A tag < dl> é usada para criar uma lista de definições. Ela consiste em pares de termos (definições) usando as tags < dt> (termo) e < dd> (definição).

Tags de formatação de caracteres:

    Tags de formatação de caracteres em HTML:

    Negrito (Bold): Use a tag < strong> para aplicar negrito ao texto.

    Itálico (Italics): Utilize a tag < em> para aplicar itálico ao texto.

    Sublinhado (Underline): A tag < u> cria sublinhado no texto.

    Imagens (Images): A tag < img> incorpora imagens.

"Links e Atributos":

(Voltar)

    Os hiperlinks são essenciais para a Web, pois permitem que documentos sejam interligados e nos conduzem de um lugar para outro na vastidão da internet. Vamos explorar como criar links em HTML e os atributos associados:

    Anatomia de um Link:

    Um link básico é criado envolvendo o texto (ou outro conteúdo) que você deseja transformar em um link dentro de um elemento . Dê ao link um atributo href (Hypertext Reference), que conterá o endereço da Web para o qual você deseja que o link aponte.

    Adicionando Informações de Suporte com o Atributo title: O atributo title fornece informações adicionais sobre o link, como o tipo de conteúdo da página ou informações importantes.

    Outros Atributos Úteis:

    Target: Define onde abrir o documento vinculado (por exemplo, em uma nova aba).

    mailto: Cria um link que abre o programa de e-mail do usuário.

    Também é possível usar imagens como links, incorporando a tag dentro da tag .

"Elementos HTML":

(Voltar)

    Os elementos HTML são as unidades fundamentais que compõem a estrutura de uma página da web. Eles definem o conteúdo e a semântica da página, fornecendo informações ao navegador sobre como apresentar e organizar o conteúdo12. Aqui estão alguns elementos HTML essenciais:

    html: O elemento raiz de um documento HTML ou XHTML. Todos os outros elementos devem ser descendentes deste elemento. Ele representa a base do documento.

    head: Fornece informações gerais (metadados) sobre o documento, incluindo o título e links para scripts e folhas de estilo.

    base: Especifica o endereço (URL) usado por todos os endereços relativos contidos no documento.

    link: Define relações entre o documento atual e recursos externos, como folhas de estilo.

    meta: Define metadados que não podem ser especificados por outros elementos HTML, como informações sobre estilos, scripts e dados auxiliares.

    style: Contém informações de estilo para um documento ou parte dele.

    title: Define o título do documento, exibido na barra de título do navegador ou na aba da página.

    Além disso, temos o elemento < body>, que representa o conteúdo principal do documento HTML. Cada documento pode ter apenas um elemento < body>. Os elementos de separação de conteúdo, como cabeçalhos e rodapés, permitem organizar o conteúdo de maneira lógica.

    Lembre-se de que esses elementos formam a base de toda a estrutura de uma página da web e são essenciais para criar conteúdo significativo e bem formatado.
